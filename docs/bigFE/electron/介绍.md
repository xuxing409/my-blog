# 介绍

## 什么是 electron？

electron 是一个使用 JavaScript、HTML 和 CSS 构建桌面应用程序的框架。通过将 Chromium 和 Node.js 嵌入到其二进制文件中，Electron 开发的应用可以跨平台的发布在 Windows、macOS 和 Linux 系统上，使用前端丰富的构建框架可以快速开发出一个高质量 UI 的桌面应用。
